Our AI Methodology

A deep dive into how our Convolutional Neural Network (CNN) works to predict blood groups.

The Process Flow

1. Dataset Collection

We curated a diverse dataset of fingerprint images, each labeled with its corresponding blood group (A, B, AB, O).

2. Pre-processing

Images were pre-processed to enhance quality. Steps included resizing, normalization, and noise reduction.

3. CNN Model Training

A Convolutional Neural Network was trained on the pre-processed data to learn unique patterns and correlations.

4. Model Evaluation

The model's performance was rigorously evaluated using metrics like accuracy, precision, and recall on a test dataset.

5. Web Integration

The trained model was deployed as a web service, enabling real-time predictions through our online demo.

Understanding the CNN Model

The core of our project is a Convolutional Neural Network (CNN), a type of deep learning model highly effective for image analysis. Unlike traditional algorithms, a CNN automatically learns hierarchical features from an image.

Our model architecture consists of several key layers:

  • Convolutional Layers: Apply filters to detect edges, textures, and patterns.
  • Pooling Layers: Reduce dimensionality, making the model robust.
  • Fully Connected Layers: Perform final classification into the blood groups.

Popular frameworks TensorFlow and Keras were used to build and train the model.
